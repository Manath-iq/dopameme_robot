# Dopameme Bot (@dopamemerobot)

## Описание
Telegram бот для генерации мемов. Основной упор сделан на **Inline-режим**, позволяющий создавать мемы прямо в чате.

## Функционал

### 1. Генерация мемов (Inline)
- **Вызов:** Пользователь пишет `@dopamemerobot` в поле ввода.
- **Шаблоны:** Бот предлагает список предустановленных шаблонов (картинок).
- **Ввод текста:**
  - Используется специальный синтаксис для разделения текста.
  - Разделитель: точка (`.`).
  - **Текст до точки:** Верхняя надпись (Top Text).
  - **Текст после точки:** Нижняя надпись (Bottom Text).
  - *Пример:* `Когда забыл деплой . А пятница вечер`
- **Стиль:** Классический мемный текст (шрифт Impact, белый цвет, черная обводка).

## Технологический стек
- **Язык:** Python 3.x
- **Фреймворк:** [aiogram v3.23.0](https://docs.aiogram.dev/en/v3.23.0/)
- **Обработка изображений:** Pillow (PIL)
- **API:** [Telegram Bot API](https://core.telegram.org/bots/api)
- **Инструменты:** Gemini CLI, MCP сервер (`context7`)
