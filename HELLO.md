# Техническое задание: Telegram-бот для создания мемов

**Цель:** Разработка бота для генерации классических мемов и демотиваторов на основе предустановленных шаблонов.

---

## 1. Приветствие и навигация
При запуске бот отправляет приветственное сообщение с описанием возможностей. Следом отправляется сообщение с интерфейсом выбора шаблона.

### Механика выбора (Карусель)
Пользователю показывается сообщение с картинкой (шаблоном) и кнопками управления.
**Структура кнопок (Inline Keyboard):**

*   **Первый ряд (Навигация):**
    *   `⬅️` (Стрелка влево): Показывает предыдущий шаблон.
    *   `✅` (Галочка): Выбирает текущий шаблон для создания **обычного мема**.
    *   `➡️` (Стрелка вправо): Показывает следующий шаблон.
    *   *Примечание:* При переключении меняется только изображение, кнопки остаются на месте.
*   **Второй ряд (Режимы):**
    *   `Демотиватор`: Выбирает текущий шаблон для создания **демотиватора**.

---

## 2. Сценарий: Создание обычного мема
**Вход:** Пользователь нажал кнопку `✅` (Галочка).

1.  Бот просит пользователя ввести текст.
2.  **Требование к вводу:** Бот сообщает пользователю формат ввода: `"текст сверху . текст снизу"`.
3.  Пользователь отправляет сообщение.
4.  Бот генерирует изображение:
    *   Используется выбранный шаблон.
    *   Текст разделяется по разделителю `.`.
    *   Первая часть накладывается сверху, вторая — снизу.
5.  Бот отправляет готовый мем пользователю.

---

## 3. Сценарий: Создание демотиватора
**Вход:** Пользователь нажал кнопку `Демотиватор` (под кнопками навигации).

1.  Бот просит пользователя ввести текст (подпись).
2.  Пользователь отправляет любой текст.
3.  Бот генерирует изображение:
    *   Используется выбранный шаблон.
    *   Изображение помещается в черную рамку (стиль демотиватора).
    *   Текст печатается под изображением.
4.  Бот отправляет готовый демотиватор пользователю.

---
*Внимание: Реализуется строго описанный функционал без добавления сторонних возможностей.*
